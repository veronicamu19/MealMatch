# MealMatch

### Rutgers meal swipe matching app
This is a platform for Rutgers students to share meal swipes with each other.  
People who need swipes and people who have extra swipes can connect on this app to make sure that no swipes are wasted.

### General Info
* written in Kotlin
* stores new users on real-time database from Google Firebase
* connects swipers and swipees by time and location
* has an option to broaden location if not enough matches are found
